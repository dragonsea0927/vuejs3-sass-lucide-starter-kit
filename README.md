<h1><img height="25" src="./public/favicon.png">&nbsp;Template Vue.js 3 + SASS + Lucide Icons</h1>


> A template to start a new project with [Vue.js 3](https://vuejs.org/), SASS and [Lucide Icons](https://lucide.dev/). ESlint and Prettier are also configured.
____
## How to develop this project

### Project setup
```bash
yarn install
```

### Compiles and hot-reloads for development
```bash
yarn serve
```

### Compiles and minifies for production
```bash
yarn build
```

### Lints and fixes files
```bash
yarn lint
```

See [Configuration Reference](https://cli.vuejs.org/config/).
